# Pandas
    Pandas is python library which helps to read and manipulate on tabular data efficiently. 
    This library is build on top of Numpy and is used by machine learning modals.
    There are two types of data exists
    1) 1 dimensional labelled data
    2) 2 dimensional labelled data
    
    Pandas refers to panel data
    
    Panda Series (1 dimensional labelled data)
    Example:
        Name
        -----
        Denny
        Mariza
        Joshua
        Jonathan
        Jacob
        Benny

    Panda DataFrame (2 dimensional labelled data)
    Example
        Name     |City
        ---------------
        Denny    |Philadelphia
        Mariza   |Philadelphia
        Joshua   |Philadelphia
        Jonathan |Philadelphia
        Jacob    |Philadelphia
        Benny    |Philadelphia
