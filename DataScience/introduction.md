# Data Science
    Data science is an interdisciplinary field that uses scientific methods, processes, algorithms and systems to extract knowledge and insights from 
    structured and unstructured data. It combines elements of mathematics, statistics and computer science, and domain expert to turn raw data into
    actionable strategy

## Core pillars of Data Science
    ### Mathematics & Statistics
       The foundation used for building models and making predictions. Techniques like linear regression, probability theory and statistical significance
       are vital for validating results.
    ### Computer Science and Programming
       Data scientists use languages like Python or R to manipulate large datasets, automate tasks, and implement machine learning algorithms.
    ### Domain Expertise
       Understanding the specific industry (e.g., finance, healthcare, or retail) is necessary to ask the right questions and interpret the data correctly

## Data Science life cycle
    ### Data capturing
       Collect row structured and unstructured data from different sources like databases, sensors, logs, web scraping
    ### Data cleaning
       In this step, data is cleaned to remove invalid data. This makes data to be usefull one for further analyzing
    ### Data Analyze
       In this step, we analyse the data and find out pattern out of it
    ### Data Visualization
       In this step, visualization methods (like reporting tools and graphs) is used to understand the visual pattern of the data.
    ### Build models and do prediction
       In this step, we build models which can make predictions related to collected data. Nowadays, we use machine learning models
       for doing the predictions
       
## Case study for application of Data Science
    Lets say we have ecommerce customer, say Amazon.com. They have a requirement that they need to analyse customers based on their behavior and interests.
    So that they can send marketing campaign to them.
    For example, they wanted to filter customer who has interest in men's clothing. So if we can filter such customers based on their buying history and
    visit history. So to do so, we collect the user history from database and then we clean it and then we analyze and generate report out of it to get the 
    list of such customers. This outcome can be used for predicting the impact on sales as well.
